关于视觉欺骗艺术馆

项目介绍

视觉欺骗艺术馆是一个交互式科普平台，致力于通过现代化的展示方式， 让公众理解人类视觉系统的工作原理和局限性。

我们结合了神经科学、心理学、数学和物理学等多个学科的知识， 为用户提供沉浸式的探索体验。终极增强版引入了 WebGL 3D 渲染、 GSAP 动画系统和游戏化学习机制，带来前所未有的视觉体验。

使命

让复杂的科学原理变得直观有趣，激发公众对认知科学的兴趣

愿景

成为最优质的视觉科学科普平台，连接科学与大众

技术栈

前端核心

  • • 原生 JavaScript (ES6+)
  • • Three.js (WebGL 渲染)
  • • GSAP (高级动画)

样式与设计

  • • Tailwind CSS
  • • 自定义 CSS 模块
  • • 现代玻璃态设计

架构模式

  • • ES6 Modules
  • • 组件化设计
  • • 数据驱动架构

性能优化

  • • 懒加载系统
  • • 多级缓存策略
  • • GPU 加速渲染
