# 📝 视觉欺骗艺术馆 - 内容创作规范

## 🎯 核心原则

### 1. 内容准确性标准

#### 科学准确性

**所有科学内容必须遵循:**

1. ✅ **同行评审来源** - 优先引用学术期刊论文
2. ✅ **权威机构认证** - 参考知名大学/研究机构资料
3. ✅ **多源验证** - 至少 3 个独立来源交叉验证
4. ✅ **时效性检查** - 确保信息未过时 (5 年内优先)

**示例:**

```markdown
❌ 不准确描述:
"这个错觉是大脑出错了"

✅ 准确描述:
"这种错觉源于视觉皮层 V1 和 V2 区域对边缘信息的处理方式。
根据 Hubel & Wiesel (1968) 的研究，方向选择性神经元对特定
角度的刺激产生反应，当多个局部线索冲突时，会产生整体感
知的偏差。"
```

#### 数据来源可靠性

| 来源类型 | 可靠性 | 使用建议 |
|---------|--------|---------|
| 同行评审期刊 | ⭐⭐⭐⭐⭐ | 优先使用 |
| 权威教科书 | ⭐⭐⭐⭐⭐ | 核心参考 |
| 知名大学网站 | ⭐⭐⭐⭐ | 可以使用 |
| 科普网站 | ⭐⭐⭐ | 需要验证 |
| 维基百科 | ⭐⭐⭐ | 初步了解，需查证 |
| 个人博客 | ⭐⭐ | 不推荐 |
| 社交媒体 | ⭐ | 禁止使用 |

### 2. 风格一致性要求

#### 写作风格

**语调:**
- ✅ 专业但不晦涩
- ✅ 生动但不轻浮
- ✅ 准确但不枯燥

**人称:**
- ✅ 使用"我们"拉近距离
- ✅ 避免过度使用"你"
- ✅ 不使用"我"

**时态:**
- ✅ 科学事实：现在时
- ✅ 历史事件：过去时
- ✅ 研究发现：现在完成时

#### 格式规范

**标题层级:**

```markdown
# 一级标题 (页面标题)
## 二级标题 (主要章节)
### 三级标题 (子章节)
#### 四级标题 (细分内容)
```

**强调规则:**

```markdown
- **加粗** - 核心概念、关键术语
- *斜体* - 外来词、首次提及的术语
- `代码` - 技术术语、变量名
- > 引用 - 重要声明、专家观点
```

**数字使用:**

```markdown
- 1-10: 使用汉字 (一个、两个)
- 10+: 使用阿拉伯数字 (15 个、100 次)
- 百分比：使用符号 (85%、96%)
- 年份：完整书写 (2024 年，非 24 年)
```

## 🏗️ "承重墙"内容识别与强化

### 什么是承重墙内容？

**承重墙内容 = 核心论点 + 关键论据 + 重要结论**

这些是文章的骨架，移除后文章将失去支撑。

### 识别方法

#### 问题测试

问自己这些问题:

1. ❓ "如果删除这段，文章的核心观点还成立吗？"
2. ❓ "这段内容是支撑论点必需的，还是锦上添花？"
3. ❓ "读者记住哪 3 个点就算读懂了？"

**如果答案是"不成立"、"必需的"、"必须记住"，那就是承重墙内容。**

#### 标记方式

**方式 1: 加粗强调**

```markdown
**彭罗斯三角揭示了人类视觉系统的一个基本特性：我们倾
向于将局部信息整合为整体，即使这种整合在物理上是不
可能的。**
```

**方式 2: 引用框**

```markdown
> **关键发现：** fMRI 研究显示，观察不可能图形时，大脑
> 的侧枕叶皮层 (LOC) 显示出异常活动模式，这表明视觉
> 系统在处理矛盾信息时的神经机制。
```

**方式 3: 专题板块**

```markdown
---
### 🔑 核心原理

**名称:** 弗雷泽螺旋错觉  
**发现者:** 詹姆斯·弗雷泽 (1908 年)  
**关键机制:** 局部方向线索影响整体感知  
**重要性:** 理解视觉系统层级处理的关键案例

---
```

## 💬 引发共鸣的内容设计

### 1. 故事化叙述

#### 结构模板

```markdown
### 一个关于 [主题] 的故事

**时间:** [具体时间]  
**地点:** [具体地点]  
**人物:** [相关人物]  
**事件:** [发生了什么]

[详细叙述故事]

**启示:** [从故事中得出的道理]
```

#### 示例

```markdown
### 埃舍尔与不可能图形的邂逅

**时间:** 1954 年  
**地点:** 阿姆斯特丹国际数学家大会  
**人物:** M.C.埃舍尔 & 罗杰·彭罗斯

当艺术家埃舍尔第一次看到数学家彭罗斯展示的"不可能三角"
草图时，他立即被深深吸引。尽管两人语言不通，但通过手势
和草图，他们进行了一场跨越艺术与科学的对话。

这次会面直接启发了埃舍尔创作其最著名的作品之一——《瀑
布》(1961)。在这幅画中，水流沿着彭罗斯三角构成的水渠永
永不停地流动，创造了艺术史上最迷人的视觉悖论。

**启示:** 伟大的创新往往发生在学科交叉的边缘地带。
```

### 2. 开放性问题设计

#### 问题类型

**反思型问题:**
- "如果你是 [人物]，你会怎么做？"
- "这个发现对你理解 [主题] 有什么启发？"

**应用型问题:**
- "你能在生活中找到类似的例子吗？"
- "如何将这个原理应用到 [场景] 中？"

**批判型问题:**
- "这个理论有什么局限性？"
- "有没有反例可以挑战这个观点？"

#### 放置位置

```markdown
## 章节末尾
- 总结后放置 1-2 个问题
- 引导读者深入思考

## 文章结尾
- 放置 2-3 个综合性问题
- 鼓励读者评论分享
```

## 🎯 内容增强技巧

### 1. 从"是什么"到"为什么"

**基础版:**
```markdown
彭罗斯三角是一个不可能图形。
```

**增强版:**
```markdown
彭罗斯三角不仅是一个有趣的不可能图形，它更揭示了人类
视觉系统的一个基本工作原理：**大脑倾向于将局部信息
整合为有意义的整体**。这种整合机制在大多数情况下帮助
我们快速理解世界，但在面对精心设计的矛盾刺激时，就会
产生这种迷人的错觉。
```

### 2. 添加具体细节

**基础版:**
```markdown
很多艺术家使用过错觉。
```

**增强版:**
```markdown
从文艺复兴时期的达·芬奇，到现代艺术大师 M.C.埃舍尔，
再到当代装置艺术家，视觉错觉一直是艺术创作的重要元
素。埃舍尔的《瀑布》(1961) 直接受到彭罗斯三角的启发，
这幅作品现藏于阿姆斯特丹市立博物馆，每年吸引超过 50
万游客前来观赏。
```

### 3. 使用对比和类比

**基础版:**
```markdown
大脑处理视觉信息很快。
```

**增强版:**
```markdown
大脑处理视觉信息的速度快得惊人。当你看到彭罗斯三角的
瞬间，视觉皮层在**130 毫秒**内就完成了初步处理——这
比你的意识察觉到它还要快。就像一台超级计算机，你的
大脑在后台默默进行了数以亿计的计算，才让你"看到"了
这个不可能的图形。
```
