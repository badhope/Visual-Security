#### 3.3.2 克莱因瓶模拟器

**科学原理**:
- 四维空间中的不可定向曲面
- 单侧曲面（只有一个面）
- 无边性（只有一条边）
- 拓扑学经典案例

**技术实现**:
```javascript

// 4D 物体在 3D 的投影
// 使用参数方程直接建模
x(u,v) = f(u,v)

z(u,v) = h(u,v)
```

#### 3.3.3 超立方体模拟器

**科学原理**:
- 四维立方体（Tesseract）
- 立体投影到三维空间
- 4D 旋转的可视化
- 高维几何入门

**技术实现**:
```javascript
// 4D 顶点（16 个）
points4D = [
    [±1, ±1, ±1, ±1] // 16 个组合
]

// 4D 旋转矩阵
// 绕 XW 平面旋转
x' = x·cos(θ) - w·sin(θ)
w' = x·sin(θ) + w·cos(θ)

// 立体投影
scale = distance / (distance - w)
x_3d = x' * scale
```

**视觉效果**:
- 🎨 16 个顶点在 4D 旋转
- 🎨 32 条边的动态连接
- 🎨 实时 60fps 渲染

### 5.3 复杂概念模拟

#### 菲欧吉利的空间（Feagin Space）

**概念说明**:
"菲欧吉利的空间"是一个抽象概念，指代非欧几里得几何空间。由于这是一个哲学/数学上的抽象概念，无法直接模拟，我们提供了：

1. **文字描述**（详细说明）:
   ```
   在非欧几里得空间中，欧几里得第五公设（平行公设）不再成立。
   
   双曲几何：过直线外一点可以作无数条平行线
   椭圆几何：过直线外一点不能作任何平行线
   
   这挑战了我们对"直线"和"平面"的直觉理解。
   ```

2. **可视化模型**:
   - 庞加莱圆盘（双曲几何模型）
   - 球面模型（椭圆几何模型）
   - 交互式参数控制

3. **数学公式**:
   ```
   双曲半径：r_hyperbolic = r * (1 - (r/R)²)
   
   庞加莱圆盘度量：ds² = 4(dx² + dy²) / (1 - x² - y²)²
   ```
