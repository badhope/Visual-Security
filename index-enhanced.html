关于视觉欺骗艺术馆

项目介绍

视觉欺骗艺术馆是一个交互式科普平台，致力于通过现代化的展示方式， 让公众理解人类视觉系统的工作原理和局限性。

我们结合了神经科学、心理学、数学和物理学等多个学科的知识， 为用户提供沉浸式的探索体验。

使命

让复杂的科学原理变得直观有趣，激发公众对认知科学兴趣

愿景

成为最优质的视觉科学科普平台，连接科学与大众

技术栈

前端框架: 原生 JavaScript (ES6+)
样式: Tailwind CSS + 自定义 CSS
架构: 模块化设计
